# React_Tutorial

### What is React?

- Javascript framework for creating user interfaces

  - Full single page web applications
  - Just certain parts of a website (like a search form)

- Component-base

- Very fast (thanks to the virtual DOM)

- Created by Facebook

  - search component
  - directory components                        ---> HTML/DOM
